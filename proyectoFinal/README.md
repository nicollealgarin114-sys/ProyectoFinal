# Proyecto Final - Sistema de Gestión Académica

Este proyecto es un sistema de gestión académica que permite administrar estudiantes, cursos e instructores.  
El sistema almacena la información usando archivos JSON y se ejecuta con una interfaz en Streamlit.

## Funcionalidades

- CRUD de Estudiantes
- CRUD de Cursos
- CRUD de Instructores
- Dashboard / panel visual
- Persistencia en archivos JSON

## Tecnologías usadas

- Python
- Programación Orientada a Objetos
- Streamlit
- JSON
